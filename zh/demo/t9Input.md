# 九宫格输入 <Badge type="tip" text="^2026.03" />

薄荷输入法内置了九宫格（T9）输入方案，主要面向 **iOS 仓输入法** 和 **元书输入法** 用户，让你在手机上也能享受薄荷词库带来的流畅输入体验。

::: info 什么是九宫格输入？
九宫格（T9）是手机上经典的输入方式：键盘只有 9 个数字键，每个键对应多个字母（如 `2` 对应 `ABC`，`3` 对应 `DEF`）。输入时按数字键，输入法根据按键组合预测候选词。
:::

## 方案特点

薄荷九宫格方案（`t9.schema.yaml`）基于薄荷全拼方案构建，继承了薄荷的核心功能：

| 特性 | 说明 |
| --- | --- |
| **词库** | 与薄荷全拼共用同一套词库（`rime_mint`） |
| **Lua 功能** | 支持时间/日期/节气、计算器等 Lua 功能 |
| **Emoji** | 支持 Emoji 输入 |
| **简繁切换** | 支持简繁体切换 |
| **分词键** | 按 `1` 键可手动分词 |

## 使用效果

### 基本输入

在九宫格键盘上，按数字键输入拼音对应的数字序列，输入法会自动匹配候选词。

例如
例如，输入「你好」：
- `你` = `nǐ` → 声母 `n`(6) + 韵母 `i`(4) → 按 `64`
- `好` = `hǎo` → 声母 `h`(4) + 韵母 `ao`(26) → 按 `426`

### 分词功能

当候选词不符合预期时，可以按 `1` 键手动插入分隔符，将输入拆分为多个独立的拼音段：

```
输入: 64 1 426
解析: ni | hao → 你好
```

### Lua 功能

九宫格方案同样支持 Lua 功能键，使用九键数字对应的字母编码即可触发：

| 功能 | 字母编码 | 九键数字 |
| --- | --- | --- |
| 时间 | `osj` | `674` |
| 日期 | `orq` | `677` |
| 农历 | `onl` | `665` |
| 星期 | `oxq` | `697` |
| 计算器 | `=表达式` | `=表达式` |

## 平台支持

九宫格方案主要在以下平台使用：

- **iOS 仓输入法**：原生支持九宫格键盘布局，体验最佳
- **元书输入法**：同样支持九宫格布局

::: warning 桌面端注意
在桌面端（Windows/macOS/Linux），九宫格方案虽然可以加载，但由于没有九宫格键盘布局，实际使用体验与全拼无异。建议桌面端用户使用薄荷全拼或双拼方案。
:::

## 激活方案

九宫格方案在薄荷中默认已激活，可以通过方案切换热键（`Ctrl/Control + ~`）切换到「中文九键」方案。

如果你的方案列表中没有九宫格，可以在 `default.yaml` 或 `default.custom.yaml` 中添加：

```yaml
# default.custom.yaml
patch:
  schema_list:
    - schema: rime_mint
    - schema: t9          # 添加九宫格方案
```

## 英文输入支持

九宫格方案默认只支持中文输入。如需支持英文，需要额外配置：

1. 将 `tools/Hamster/melt_eng.custom.yaml` 复制到配置目录
2. 在 `t9.schema.yaml` 的 `engine/translators` 中取消注释英文翻译器

```yaml
# t9.schema.yaml（或使用 custom 文件覆写）
engine:
  translators:
    - table_translator@melt_eng   # 取消此行注释以启用英文输入
```

::: tip 仓输入法用户
`tools/Hamster/` 目录下已经提供了针对仓输入法优化的配置文件，直接使用即可。
:::
