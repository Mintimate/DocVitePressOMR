# Oh My Rime 支持的输入方案

## 默认激活的方案

- **薄荷拼音 (全拼)** (`rime_mint`) [全拼]: 招牌主打输入类型，支持各种反查、中英混合输入，词库基于雾凇拼音词库。
- **地球拼音** (`terra_pinyin`) [全拼]: 带声调输入的拼音方案。
- **小鹤双拼** (`double_pinyin_flypy`) [双拼]: 基于小鹤双拼键位图定制，支持辅码滤镜提示。
- **98五笔 (薄荷定制)** (`wubi98_mint`) [五笔]: 基于98五笔基础版本定制的轻量版本。
- **86五笔 (极点码表)** (`wubi86_jidian`) [五笔]: 基于86五笔极点码表定制的轻量版本。

## 可用但未默认激活的方案

- **自然码双拼** (`double_pinyin`) [双拼]: 基于自然码键位图定制的双拼输入方式。
- **智能ABC双拼** (`double_pinyin_abc`) [双拼]: 基于智能ABC键位图定制的双拼输入方式。
- **微软双拼** (`double_pinyin_mspy`) [双拼]: 基于微软双拼键位图定制的双拼输入方式。
- **搜狗双拼** (`double_pinyin_sogou`) [双拼]: 基于搜狗双拼键位图定制的双拼输入方式。
- **紫光双拼** (`double_pinyin_ziguang`) [双拼]: 基于紫光双拼键位图定制的双拼输入方式。

## 如何激活其他方案？

如需开启未默认激活的方案，请参考文档: https://www.mintimate.cc/zh/guide/defaultActivationScheme.html
