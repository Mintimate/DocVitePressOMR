## 开发者指南

如果你想参与文档的编写，克隆本仓库，然后执行以下命令： 
```bash
# 安装依赖
yarn install
# 运行
yarn dev
# 打包(输出到 .vitepress/dist)
yarn build
```

如果你想参与薄荷配置的编写，可以前文：
- [oh-my-rime: https://github.com/Mintimate/oh-my-rime](https://github.com/Mintimate/oh-my-rime)
